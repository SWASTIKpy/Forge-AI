# Forge AI

Forge AI is an AI-powered full-stack application builder inspired by Loveable. Describe the application you want to build in natural language, and Forge AI generates a complete project with a live preview, editable source code, automatic dependency management, and AI-assisted debugging.

---

## Features

- Generate full-stack applications from natural language
- Live application preview while code is being generated
- Built-in code editor and file explorer
- Automatic installation of required frameworks and dependencies
- Sandboxed execution environment for safe builds and runtime error detection
- AI-powered error fixing using runtime errors and project context
- Version history for every AI generation and bug fix
- Restore or compare previous generations
- Export projects as a ZIP for local development

---

## Tech Stack

- Next.js
- TypeScript
- Supabase
- Tailwind CSS
- shadcn/ui
- Cline

---

## How it Works

1. Describe the application you want to build.
2. Forge AI plans and generates the project.
3. Dependencies are installed automatically.
4. The application runs in a sandbox with a live preview.
5. Runtime errors are analyzed and fixed using AI.
6. Every generation is saved and can be restored.
7. Export the finished project as a ZIP and continue development locally.

---

## Running Locally

```bash
git clone https://github.com/yourusername/forge-ai.git

cd forge-ai

npm install

npm run dev
```
